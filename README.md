# uboot
uyibai basic project for Spring Boot By uyibai

基于[jeecg](http://doc.jeecg.com)开发

#### SpringBoot 单服务开发项目基础架构

##### 包含内容

###### 1. 基础内容
- [x] Swagger 接口文档
- [x] 自定义异常、全局异常处理
- [x] MybatisPlus集成、代码生成工具
- [x] 接口返回内容封装
- [x] log4j日志配置
- [ ] jetcache集成、配置

###### 2. 功能列表
- [ ] shiro 权限控制、登陆、权限管理
- [ ] 文件上传管理
